# SHE shares メディア戦略転換プラン

## エグゼクティブサマリー

**現状**: 1月CV実績6件（目標44件）、達成率13.6%。このままでは3月100件は不可能。

**発見**: 300本以上のインタビュー記事と検索ページ（experiencesearch）が存在するが、**全く活用されていない**。

**戦略**: 今あるアセットを最大活用し、「ロールモデルが見つかるメディア」へ転換。短期CVと長期ブランディングを同時に実現する。

---

## 1. なぜ今のやり方では無理なのか

### 現状の問題構造

```
【今の導線】
記事を読む → 「無料体験レッスンを受ける」→ 申込ページ

【ユーザーの心理】
「私みたいな人でも大丈夫かな...」
「子供いても続けられる？」
「本当に転職できた人いるの？」
　　　　↓
不安が解消されないまま離脱
```

### 数字が示す事実

| 指標 | 通常キーワード | 女性系キーワード | 差 |
|------|---------------|-----------------|-----|
| CTR | 0.61% | 1.03% | **1.7倍** |
| 順位 | 10.41 | 9.69 | +0.72位 |

**女性系キーワード（ママ・主婦・女性）で検索する人は、より「自分と同じ境遇の人」を探している。**

→ だからCTRが高い。だからここにロールモデル訴求が刺さる。

> **自分と同じ境遇から人生を変えた先輩を見つける場所**である。

## 3. 短期CV最大化プラン（1〜3月）

### Phase 0：今週中（1/19〜1/26）

**やること：experiencesearchを機能させる**

**優先する属性グループ（女性系KWと親和性が高い順）：**
1. ママ × 転職成功
2. ママ × 副業・フリーランス
3. 20代 × 未経験から転職

5. 40代 × 新しい挑戦

### Phase 1：1月末まで（1/27〜1/31）

**やること：CTAを全面切り替え**

#### Before
```
CTA：「無料体験レッスンを受ける」
　　　　↓
　　 申込ページ
```

#### After
```
CTA：「あなたに近い先輩を探す」
　　　　↓
　　 experiencesearch（属性検索）
　　　　↓
　　 該当するインタビュー記事一覧
　　　　↓
　　 記事を読む
　　　　↓
　　「この先輩の話を聞いてみる」
　　　　↓
　　 日程選択LP
　　　　↓
　　 体験レッスン申込完了（CV）
```

#### 実装優先順位

1. **女性系キーワード記事から着手**（CTRが1.7倍なので効果検証しやすい）
2. **キラーコンテンツのCTAを変更**
3. **全記事のCTAを順次変更**

### Phase 2：2月（2/1〜2/28）

**やること：効果検証 + 体験レッスン改善**

#### 検証指標

| 指標 | 1月実績 | 2月目標 |
|------|--------|--------|
| CTR（CTA→experiencesearch） | - | 5%以上 |
| 検索→記事閲覧率 | - | 30%以上 |
| 記事→体験申込率 | - | 3%以上 |
| CV数 | 6件 | 69件 |

#### 体験レッスン改善（2パターン）

**【パターンA】2部構成がいける場合**

```
【新・体験レッスン】90分構成

前半45分：ロールモデルとの座談会
- ユーザーが選んだ属性グループの卒業生が登場
- 「私も最初は不安でした」から始まるリアルな体験談
- 質疑応答（子供いても大丈夫？収入は？など）

後半45分：レッスン体験
- 実際のカリキュラムを体験
- 入会案内
```

**メリット：**
- 「話を聞く」ハードルが低い
- 不安解消 → レッスン体験の流れで自然にCVへ
- 卒業生にバックマージンで報酬を支払う仕組みで持続可能

**【パターンB】2部構成がいけない場合**

```
【代替案】インタビュー記事の強化

① 記事内に「この先輩に質問を送る」フォーム設置
　→ 非同期で回答（記事に追記 or メール返信）

② 記事末尾に「この先輩と同じ体験レッスン」CTA
　→ その卒業生が受けたコースの体験へ直接誘導

③ 月1回「卒業生オンライン座談会」開催
　→ 記事で興味を持った人向けのイベント
　→ 体験レッスンとは別枠で実施
```

**メリット：**
- 体験レッスンの運用を変えなくて済む
- 記事側の改善だけで完結
- ただしCVへの直接効果はパターンAより弱い

### Phase 3：3月（3/1〜3/31）

**やること：全記事展開 + 最適化**

- 全記事のCTAを「ロールモデル訴求」に統一
- 効果の高い属性グループを特定し、インタビュー記事を追加作成
- 体験レッスンの2部構成を定着させる（パターンAの場合）

#### 3月目標達成シミュレーション

```
セッション：90,000（現状トレンド維持）
CTR（CTA→検索）：5%（新CTA効果）
検索→記事閲覧：30%
記事→体験申込：3%

90,000 × 5% × 30% × 3% = 40.5件

＋ 直接申込（従来導線残し）：30件
＋ 座談会イベント経由：30件

合計：約100件 ✅
```

## 4. 長期メディアコンセプト設計

### 今のSHE sharesが抱える課題

#### 課題1：差別化できていない
- 「職種軸（Webデザイン・ライティング等）」での集客は他スクールと同じ
- 「女性向け」だけでは競合と差別化できない
- Famm、デジハリ、テックキャンプ等と比較されて選ばれない

#### 課題2：ユーザーが意思決定できない
- 「私みたいな人でも大丈夫？」に答えられていない
- 自分の生活にどう当てはめられるかイメージできない
- パーソナライズされた情報がない

#### 課題3：アセットが活かされていない
- 300本以上のインタビュー記事 → 埋もれている
- experiencesearch → 存在するが機能していない
- 卒業生ネットワーク → 活用されていない

**SHE sharesには、すでに「ロールモデルメディア」になるための材料が全部揃っている。**

- 300本以上のインタビュー記事
- 属性タグ
- 検索ページ
