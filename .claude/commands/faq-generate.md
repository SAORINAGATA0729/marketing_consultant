# FAQ自動生成

質問リストからLLMO対策されたFAQ回答を自動生成する。

## 寺倉メソッド（参照元）
- 1/15 KAAAN/tantan確認
- 1/21 コンテンツSEO｜寺倉×青木

## 実行手順

### 1. インプットを確認
ユーザーに以下を確認：
- 質問リストのファイルパス（CSV or マークダウン）
- サービス名（例：giftee for Business）
- サービス固有情報があれば（任意）

### 2. 質問リストを読み込む
指定されたファイルから質問を抽出。

### 3. FAQ回答を生成
各質問に対して、以下のルールで回答を生成：

#### 回答ルール（寺倉メソッド）

**構造：**
```
【結論】〇〇です。

【具体】
1. 〇〇
2. 〇〇
3. 〇〇

【締め】〇〇
```

**ルール：**
- **結論ファースト**: 最初に答えを書く
- **300文字程度**: 簡潔に
- **PR表現禁止**: サービスの宣伝文句は入れない
- **サービス紹介禁止**: 「〇〇は素晴らしいサービスです」等は不要
- **AIに最適な形式**: GPT/Geminiが返すような構造で書く

**NGワード（使用禁止）：**
- 「業界最高水準」「画期的な」「革新的な」
- 「多くのお客様にご愛用」「〜様にご好評」
- 「ぜひお試しください」「お気軽にお問い合わせ」

### 4. 結果を出力
以下の形式でファイル出力：

**マークダウン形式：**
```markdown
# FAQ一覧

## Q1: [質問]
[回答]

## Q2: [質問]
[回答]
```

**CSV形式：**
```csv
質問,回答
"〇〇とは？","〇〇です。..."
```

### 5. 確認プロセス（任意）
生成後、ユーザーが希望すれば：
- 同じ質問をGPT/Geminiに投げて比較
- 差異があれば調整

## 使用例

```

ユーザー: giftee for Business

```
