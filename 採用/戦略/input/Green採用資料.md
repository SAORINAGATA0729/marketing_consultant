# サービス概要資料：Green

## コンセプト・ビジョン
**「転職者にとって自分だけの青信号 (Green Light) ＝ 生き生きと働ける環境」を見つけられるサービスでありたい** という想いがGreenの名前の由来です。
Greenが描く世界は、採用した方が活躍することで企業が成長することや、就業した転職者がその企業で働くことを通して人生がより豊かになることが重要だと考えており、「人と組織を元気にする」というVisionを掲げています。

## 市場背景
昨今の中途採用市場における採用難易度は年々上がり続けております。中でもIT/Web職種の求人倍率は10倍近くを維持しており、多くの企業が限られた人材を採用することに様々な課題を感じています。

そんな状況においてもGreenは最適なマッチングを実現するために、各社の魅力を求職者に届けること、求職者の方が企業のことを深く知れることを重要視し、今までにないマッチングを生み出す転職プラットフォームを運用しております。
サービス開始の2006年からこれまで約9,600社を超える企業様にご利用いただくともに、120万人を超える求職者の転職活動を支援しております。

# 03 Greenの特長

### 特長①：120万人を超えるユーザーへ自由にアプローチ
転職意欲や条件面のフィルターを掛けることが可能で、希望条件に一致するユーザーを自由に探すことができます。
1クリックで100人にアプローチする機能も用意されており、採用のご経験やノウハウが無い方でも簡単に採用活動を始めることができます。
*   毎月の新規登録人数：1.8万人以上
*   一括アプローチ：上限無し
*   個別スカウト：毎月1,000通無料

### 特長②：企業の魅力を伝えるブランディングページ
求人票だけでなく、各社オリジナルの企業紹介ページを用意しています。ユーザーに自社の魅力をしっかりと伝えることで、想いに共感したターゲットからの応募に繋がります。
*   プロのライターが約10,000文字で記事取材・作成
*   プロのフォトグラファーが約40枚の写真を撮影

### 特長③：採用コストを大幅に削減
一度ご参画いただければ、その後無料で継続利用が可能です。ランニングコスト0円で、費用の掛け捨てリスクもございません。
*   求人追加/編集も「無料無制限」
*   年収に関わらず成功報酬が一律固定

**【コスト比較例：年収500万円の人材を1年間で6名採用の場合】**
*   **人材紹介:** 約1,050万円（500万×紹介料35%×6名）
*   **他社DRサービスA:** 約1,110万円（年間利用料60万＋500万×紹介料35%×6名）
*   **Green:** **890万円**（初期費用90万＋成功報酬120万×6名 ※スタンダードプラン想定）→ **約200万円のコストカット！**

# 04 料金体系
Greenの料金体系は **「初期費用 ＋ 成功報酬費用」** です。

### プラン一覧（初期費用）
全てのプランで掲載期間は無制限です。

| プラン名 | 初期費用 | 含まれる主な機能 |
| :--- | :--- | :--- |
| **ライト** | 60万円 | 基本機能のみ |
| **ベーシック** | 75万円 | PRタブ or インタビュータブ |
| **スタンダード** | 90万円 | PRタブ、インタビュータブ |
| **プレミアム** | 120万円 | PRタブ、インタビュータブ、求人検索上位表示(×3回)、メルマガ(×2回) |
※更新費用は一切不要。お支払いは初回参画時のみ。

**【全プラン共通の基本機能】**
*   Greenのデータベースから求職者を検索（求人数無制限）
*   スカウト毎月1,000通無料
*   「気になる」機能は無制限
*   企業ブランディング用の記事作成＋写真撮影（企業詳細ページの作成）

### 成功報酬費用（職種ごとに一律固定）
プランに関わらず、採用した職種によって固定の成功報酬が発生します。

| 職種分類 | 成功報酬額 |
| :--- | :--- |

| エンジニア・技術職（電気/電子/機械/半導体） | 60万円 |

### オプション機能・その他
初期費用に加え、ご希望に合わせてカスタマイズが可能です。

*   **採用支援プラン（20万円）**: カスタマーサクセス担当によるプランニング、記事確認、スカウト文作成、月次振り返り（計4回程度の打ち合わせ）。

*   **求人検索上位表示（40万円）**: 2週間、求人検索結果の上位に表示（3求人まで）。
*   **メールマガジン（10万円）**: 登録ユーザーへ1回配信。
*   **求人特集（10万円）**: 特集ページに1週間掲載（3求人まで）。
*   **求人記事取材作成（10万円）**: プロのライターによる求人作成（3求人まで）。
*   **プレミアムインタビュー（260万円）**: Green内での一社独占記事公開（6ヶ月）。
*   **スカウト追加（40万円）**: スカウト200通追加（購入月の翌月末まで有効）。
*   **二次利用権利（20万円）**: 写真・記事データを自社HP等で利用可能（各10万円）。

# 05 お客様の声・実績
Green掲載後3ヶ月以内の内定率は75%以上です。推奨する活用方法（会いたい送信10件以上、スカウト送信50通以上、気になる送信300件以上）を実践すると内定率アップに繋がります。

*   **エンジニア採用が目的で導入したが、別職種の採用にも繋がった**
    ハイクラスエンジニアや若手、デザイナー、営業・バックオフィスなど複数職種で直近数年で20人程の採用に成功。
*   **創設2年目で1年間に4名の採用を実現**
    普段エージェントでは紹介してもらえないような優秀な人材を採用できた。アプローチの利用期限・制限が無い点が魅力。
*   **認知度の低い企業でも戦える**
    「気になる」機能を使えば一度に100名に会社を認知してもらえる。1日10分の運用だけで数十名の採用を実現。

# 06 よくあるご質問

**Q. 掲載はいつから出来ますか？**
A. 取材など含めお申し込みから3〜4週間で掲載ができます。それより早い掲載をご希望の場合は『簡易掲載』という形で最短2日で掲載が可能です。

**Q. IT系以外の職種は採用できないのでしょうか？**
A. IT系の職種以外も採用可能です。登録人数的に採用しやすいのはエンジニア、デザイナー、営業。採用倍率的に採用しやすいのは、マーケティングやバックオフィスの人材です。

**Q. ブランディングページの取材はオンラインですか？**
A. オフライン、オンラインともに対応可能です。
