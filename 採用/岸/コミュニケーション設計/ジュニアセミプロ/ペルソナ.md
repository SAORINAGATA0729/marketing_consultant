    - 事業会社でマーケティングや事業開発に携わる
    - オペレーション構築、組織立ち上げ、メディア運営など、多岐にわたる業務を経験
    - 一定の成果を出してきたが、次のステップを模索している
- **志向性**:
    - 事業会社での経験を活かしつつ、コンサルタントとして幅広い事業課題に取り組みたい
    - 「何者か」を見つけたいという漠然とした不安がある
    - 明確なビジョンやWillはないが、成長意欲は高い

### 3. 仕事における価値観・行動原理

- **成長意欲**:
    - 現状の環境で「やりきった」感がある一方で、次のステップを模索している
    - 「何者か」を見つけたいという漠然とした不安がある
    - 明確な目標像はなく、現状への漠然とした不安の方が強い
- **事業への貢献意欲**:
    - 事業会社での経験を活かしつつ、コンサルタントとして幅広い事業課題に取り組みたい
    - 「自分がどうなれるか」が見つかる環境を求めている
- **年収への意識**:
    - 年収600-700万円台で、成長機会と適切な報酬を両立したい
    - 年収が下がることは避けたい（成長できる環境でも、年収が下がる場合は応募しない）

### 4. 抱えている葛藤とNext Stepへの動機

**「現在の状況」**
- 事業会社で一定の成果を出し、「やりきった」感がある
- マネジメント経験も積み、ある程度の自信を得ている
- しかし、「このままで私何が残るんだろう」という不安を感じ始める
- 「何者か」を見つけたいという漠然とした不安がある

**「転職を考える動機」**
- **転職意欲が高い**: Wantedlyやビズリーチなどの転職媒体を積極的に活用している
- **成長機会と適切な報酬を両立したい**: 年収が下がることは避けたい
- **「自分がどうなれるか」への答え**: 明確な目標像はなく、現状への漠然とした不安の方が強い。自分がどうなれるかがわからない状態なので、そこに対してアンサーがあるような環境を求めている
- **コンサルタントへの関心**: 事業会社での経験を活かしつつ、コンサルタントとして幅広い事業課題に取り組みたい

### 5. KAAANへの応募・マッチング仮説

**なぜKAAANなのか（応募ストーリー）**
1.  **Wantedly経由での認知**: WantedlyでKAAANの募集を見つける。スカウトメッセージをもらい、「オープンポジションで未経験でもチャンスあるんだ」と知る。
2.  **「オープンポジション」の概念**: 明確な職種を募集しているわけではなく、一緒に形にしていくという姿勢が、「何者でもなかった」状態に響く。
3.  **「総合的にやっている」という表現**: 「一つの手段に取られるとかではなく、いろいろできるのか、いろんな形で学べるのかな」という期待に応える内容。
4.  **成長機会と適切な報酬**: 年収が下がらない、または成長機会と適切な報酬を両立できる環境であることが重要。

**採用コミュニケーションのポイント**
- **訴求メッセージ**: 「整った環境より、あなたの手で道を切り拓く環境へ。ユーザーのために本質を追求する、終わりのない0→1を一緒に。」
- **評価すべき点**: スキルセット以上に、マインドセット（環境適応力、グリット力、誠実さ）を高く評価すべき。
- **採用戦略**: **ダイレクトリクルーティングやWantedly運用による直接的なアプローチ**が有効。Wantedlyの「話を聞きに行きたい」機能を活用し、カジュアル面談から始める。

### 6. 年収・待遇への考え方

- **年収は下げたくない**: 600-700万円台の場合は、成長できる環境でも、年収が下がる場合は応募しない
- **成長機会と適切な報酬を両立したい**: 年収が上がる、または成長機会と適切な報酬を両立できる環境であることが重要
- **Wantedlyで年収が表示されない**: Wantedlyで年収が表示されないため、他の転職サイト（リクナビ、Greenなど）で年収情報を確認する
