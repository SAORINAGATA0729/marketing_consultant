## ペルソナ詳細：成長と挑戦を求める「次世代コンサル候補」

### 1. 基本プロフィール

| 項目 | 内容 |
| :--- | :--- |
| **年齢** | 27-28歳前後 |
| **現職** | 事業会社 / マーケティング・事業開発 |
| **経験年数** | 5-6年 |
| **年収** | 500-600万円台 |
| **学歴** | 大学卒業 |
| **役割** | マーケティング、事業開発、オウンドメディア運営、CRM構築など |

### 2. キャリア・スキルセット

- **実務経験**:
    - 事業会社で5-6年間、マーケティングや事業開発に携わる
    - **オウンドメディア立ち上げ・運営（メイン）**: 新規メディアの立ち上げから、SEO重視のコンテンツ制作、グロースまでを一通り経験。はじめはGoogle検索になんにもひっかからなかった無名メディアを、メーカーやキャリアの方々、そしてユーザーに認知される規模まで成長させた経験
    - **CRM構築（メイン）**: ECサイトのCRMディレクターとして新たにチームも設立し、CS調査や顧客分析を重ねる中で、「オウンドメディアをどう活かすか？」という視点で物事を考える経験
    - **オペレーション構築（サブ）**: 業務フローの改善や効率化など、オペレーション面での改善経験もあるが、メインの強みではない
    - **マネジメント経験**: チームリーダーとして、メンバーの育成や組織運営を経験
    - 社内で評価される成果を出し、一定「やりきった感」がある状態。次のステップを模索している
- **志向性**:
    - **支援会社でも事業会社でも良い**: 職種や会社の形態にはこだわらず、「自分のやってきたことが活かせそう」「いま自分にないものが身につけられそう」という環境を求めている
    - **「おもしろい人たちとおもしろいことをしたい」**: 明確なビジョンやWillはないが、「おもしろい人たちとおもしろいことをしたい、し続けたい」という気持ちがある
    - **新しい環境に飛び込むのが好き**: 「新しいもの好き」で、新しい環境に飛び込むことを厭わない
    - 0→1の環境での挑戦を好み、「道なき道を作る」ことにやりがいを見出す

### 3. 仕事における価値観・行動原理

- **成長意欲**:
    - 現状の環境で一定「やりきった感」がある状態。次のステップを模索している
    - 明確な目標像はないが、「おもしろい人たちとおもしろいことをしたい、し続けたい」という気持ちがある
- **事業への貢献意欲**:
    - 事業会社での経験を活かしつつ、コンサルタントとして幅広い事業課題に取り組みたい
    - 「自分がどうなれるか」が見つかる環境を求めている
    - **「自分のやってきたことが活かせそう」「いま自分にないものが身につけられそう」**: 支援会社でも事業会社でも、この2つの条件が揃う環境であれば興味を持つ
- **年収への意識**:
    - 年収500-600万円台で、成長機会と適切な報酬を両立したい
    - 年収が下がることは避けたい（成長できる環境でも、年収が下がる場合は応募しない）

### 4. 抱えている葛藤とNext Stepへの動機

**「現在の状況」**
- 事業会社で5-6年間、オウンドメディアの立ち上げ・運営、CRM構築など、具体的な成果を出してきた
- マネジメント経験も積み、社内での評価も確立している
- 一定「やりきった感」がある状態。達成感と自信はあるが、次のステップを模索している
- オウンドメディアやCRMの経験はあるが、「このままで私何が残るんだろう」という不安を感じ始める

**「転職を考える動機」**
- **転職意欲が高い**: Wantedlyやビズリーチなどの転職媒体を積極的に活用している。Wantedly経由でスカウトメッセージをもらうこともある
- **成長機会と適切な報酬を両立したい**: 年収が下がることは避けたい
- **「自分がどうなれるか」への答え**: 明確な目標像はないが、「おもしろい人たちとおもしろいことをしたい、し続けたい」という気持ちがある。自分がどうなれるかがわからない状態なので、そこに対してアンサーがあるような環境を求めている
- **支援会社でも事業会社でも良い**: 「自分のやってきたことが活かせそう」「いま自分にないものが身につけられそう」という2つの条件が揃う環境であれば、支援会社でも事業会社でも興味を持つ
- **「おもしろい人たちとおもしろいことをしたい」**: 具体的な目標はないが、「おもしろい人たちとおもしろいことをしたい、し続けたい」という気持ちがある
- **新しい環境に飛び込むのが好き**: 「新しいもの好き」で、新しい環境に飛び込むことを厭わない性格
- **事業へのインパクト**: 単なる「業務遂行」に留まらず、事業全体の数字やユーザー体験に直結する仕事を求めている

### 5. KAAANへの応募・マッチング仮説

**なぜKAAANなのか（応募ストーリー）**
1.  **Wantedly経由での認知**: WantedlyでKAAANの募集を見つける、またはWantedly経由でスカウトメッセージをもらう。「一度、お話ししませんか？」というメッセージに「新しいもの好き」な自分はすぐに会うことを決意。
2.  **「自分のやってきたことが活かせそう」**: オウンドメディア立ち上げ・運営、CRM構築などの経験が活かせそうだと感じる。
3.  **「いま自分にないものが身につけられそう」**: 支援会社として様々な事業課題に取り組むことで、自分のスキルをより幅広く活かせそうだと感じる。
4.  **「オープンポジション」の概念**: 明確な職種を募集しているわけではなく、一緒に形にしていくという姿勢が、「何者でもなかった」状態に響く。
5.  **「総合的にやっている」という表現**: 「一つの手段に取られるとかではなく、いろいろできるのか、いろんな形で学べるのかな」という期待に応える内容。
6.  **「おもしろい人たちとおもしろいことをしたい」**: 面談で「こんな働き方があるのか！」とわくわくし、「おもしろい人たちとおもしろいことをしたい」という気持ちに応える環境だと感じる。
7.  **成長機会と適切な報酬**: 年収が下がらない、または成長機会と適切な報酬を両立できる環境であることが重要。

**採用コミュニケーションのポイント**
- **訴求メッセージ**: 「整った環境より、あなたの手で道を切り拓く環境へ。ユーザーのために本質を追求する、終わりのない0→1を一緒に。」
- **評価すべき点**: スキルセット以上に、マインドセット（環境適応力、グリット力、誠実さ）を高く評価すべき。
- **採用戦略**: **ダイレクトリクルーティングやWantedly運用による直接的なアプローチ**が有効。Wantedlyの「話を聞きに行きたい」機能を活用し、カジュアル面談から始める。

### 6. 年収・待遇への考え方

- **年収は下げたくない**: 500-600万円台の場合は、成長できる環境でも、年収が下がる場合は応募しない
- **成長機会と適切な報酬を両立したい**: 年収が上がる、または成長機会と適切な報酬を両立できる環境であることが重要
- **Wantedlyで年収が表示されない**: Wantedlyで年収が表示されないため、他の転職サイト（リクナビ、Greenなど）で年収情報を確認する
