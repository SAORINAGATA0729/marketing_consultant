# 応募ジャーニー：コンサルタント候補（エグゼクティブクラス）

## エグゼクティブサマリー

**コンセプト：Seek for "The Hardest"（最も困難で、最も成長できる場所へ）**
エグゼクティブクラスのコンサルタント候補は安定や待遇ではなく、**「自分の成長角度を最大化できる環境」**を求めている若き野心家である。
**転職意欲は低く、半年〜1年かけてじわじわと高まっていく**タイプ。Wantedlyやビズリーチなどの転職媒体は使わず、MOサイトやメルマガ、オフラインイベントなど、**ブランディング戦略によるロングタームアプローチ**が有効。
コンサルという職種はあくまで手段。本質的な動機は、**「整った環境の天井」を突き破り、「プロフェッショナルな猛者たち」の中で自分の実力を証明すること**にある。
KAAANは彼らにとっての「就職先」ではなく、自らを鍛え上げるための**「道場（Colosseum）」**として映るべきである。

- これまで転職やクリスを考えたことが1回もなかった

## 重要な発見

### 転職意欲が低い層へのアプローチ

1. **Wantedlyやビズリーチなどの転職媒体は使わない**
   - 転職意欲が低いため、転職媒体には登録しない
   - Meetyやユートラなど、カジュアル面談やイベント目的のプラットフォームは使う

2. **ブランディング戦略によるロングタームアプローチが必要**
   - 半年〜1年かけてじわじわと転職意欲が高まっていく
   - タイミングでアプローチするのではなく、徐々に網を張る感じ
   - MOサイト、メルマガ、コーポサイト、X、Meety、ユートラ、オフラインイベントなどでの情報発信が重要

3. **年収は下がってもよいが、600-700万円台だと違う判断**
   - 900万円台の場合は、投資期間として年収が下がってもよいと判断
   - 600-700万円台の場合は、ジャンプアップさせたいという気持ちが強く、年収を下げたくない

4. **明確なビジョンやWillはない**
   - 須藤さんと同様、「何者かになりたい」という漠然とした不安がある
   - 明確な目標像はなく、現状への漠然とした不安の方が強い
