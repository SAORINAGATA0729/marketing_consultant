# １．オペレーター（人物イメージ：須藤萌）

### ポジションの位置づけ

クライアントワークにおいて、コンサルタント（フロント）の右腕として、
AI／ノーコード／ワークフロー設計の力を活かし「再現性あるマーケティング運用」を形にする役割。
マーケ/AIの高度な知識は入社時点では不要。吸収力・素直さ・再現性のある実行力が最重要。

### ミッション

・フロントの意図を正しく言語化し、オペレーションに落とし込む
・AI編集部の「仕組み化・自動化・標準化」を推進
・情報整理やワークフロー整備を通じて、品質とスピードを両立する環境をつくる

### 業務内容（抽象化）

・プロンプトチェーン／ワークフローの設計・構築・チューニング
・Notion、n8n、各種ノーコードツールを用いた業務設計・改善
・制作プロセスの標準化と運用改善
・コンサルタントからの指示を正しく解釈し、再現性のある形に翻訳
・軽度のコンテンツ制作（下書き、要素抽出、整形）

### 求める資質（“須藤萌”の抽象化）

・素直に吸収し、教わったことをそのまま正確に実行できる
・現場の負担や温度感を理解できる “バランス感覚”
・コミュニケーションが円滑で、誰とでも摩擦なく協働できる
・効率化だけでなく「ユーザー側の心地よさ」も自然と考えられる
・こだわりの強すぎない柔らかさ（調整力・適応力）
・構造的に物事を整理することが得意、または苦手ではない

### 必須条件

・業務フロー／タスクの整理が苦手ではない
・Googleスプレッドシート、Notionなど基本的なITリテラシー
・コミュニケーションを伴う業務経験（職種不問）

### 歓迎条件

・業務改善／オペレーション改善の経験
・ノーコードツール（Notion, n8n, Zapierなど）利用経験
・コンテンツ制作の基本経験
・マーケティング領域への興味関心

### 想定する人物像ペルソナ

・「器用貧乏」とよく言われるが、現場に入ると異様に強い
・新しい環境に溶け込むのが速く、情報吸収が早い
・ルールや仕組みづくりが得意、または好き
・スペシャリストより“現場の要”として全体を回す仕事を好む

### 1. 役割・ポジション
- 主に「オペレーション寄りのコンテンツ制作支援」。
- 編集/ディレクション専任ではなく、業務フロー設計・運用改善・効率化の実務に強み。
- チームや現場全体を巻き込み、AIやノーコードツールを使いながら仕組み化・自動化を推進。

### 2. 業務内容
- コンテンツ企画・制作・拡張の基盤や仕組みを設計し運用までをカバー。
- 業務フローの再現性や効率・品質担保を目指す。現場の負担を抑えつつ情報を集め、ワークフローを整理。
- 具体的にはNotionやN8Nなどで業務の標準化・自動化を推進。

### 3. 経歴・経験
- マイベスト（前職）ではオペレーションチーム→事業推進部で活躍。
- 画像制作の業務フロー構築と生産性・品質向上を達成。
  - 10枚/週→350～400枚/週に拡大しつつコスト・品質も最適化。
- 理想のアウトプットを分解し現場に落とし込み、再現性のある運用フローを作れる。

### 4. 性格・特徴
- バランス型で、人や状況に合わせてそつなく業務を遂行する「器用貧乏」タイプ。
- ESFP（エンターテイナー型傾向）：明るくポジティブ、柔軟、周囲とのコミュニケーション・協力も得意。
- 効率一辺倒になりすぎず、ユーザーや現場目線も重視する。
- 新しいツールや業務も素直に吸収。こだわりや自我が強すぎず、組織や現場に溶け込みやすい。

### 5. スキルセット
- 業務オペレーション設計／デジタルワークフロー自動化
- コンテンツ制作（基本～中級）
- 各種ノーコード／業務効率化ツールのキャッチアップ力
- 高いコミュニケーション／現場巻き込み力

### 6. キャリア志向・転職先決定のポイント
- 「業務の効率化や仕組み化」「現場のリアルな声を吸い上げる体制づくり」に価値を感じ、その中で自分の強みを発揮できる環境を重視。
- 尖ったスペシャリストよりも、全体最適や調整型・改善推進型の仕事を好む傾向。
- 自分が仕組み作りや推進役として裁量をもって貢献できる環境、および現場の声がちゃんと生きる組織を志向。
- 転職先の決め手は「マーケティング」「業務改善」領域への関心・経験が活かせること、組織の雰囲気やフィーリング、一緒に働く人との相性やコミュニケーションも重要視している。
- 「マーケティング経験」は本人の中でキーワードになっており、採用側の声掛け（Wantedly経由）もこの文脈だった。
