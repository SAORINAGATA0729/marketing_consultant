# 須藤さんヒアリング結果を踏まえた改修案

## ヒアリングで判明した重要なポイント

### 1. ペルソナに関する修正点

#### ❌ 修正が必要な点
- **「独立採算制度が決め手」**: 面白い制度だと思ったが、自分がエージェンシーとして動くわけではなかったので、それほど大きな決め手ではなかった
- **「明確な目標像」**: 明確な目標像はなく、現状への漠然とした不安の方が強かった

#### ✅ 正しい理解
- **「何者でもなかった」状態**: ビジネスパーソンとしてどこまでできるのか、何ができるのかわからないという漠然とした不安
- **「このままで私何が残るんだろう」**: 4年間勤めてある程度経験した・やりきった感があったところで「ここからどうしよう」と考えた時に感じた不安
- **「目指すべきもの」より「現状への不安」**: 目指すべきものというよりかは、このままでいいのか、なんとなく不安だけど、でも目指す先もよくわかってないからどうしようみたいな状態

### 2. ジャーニーに関する修正点

#### ❌ 修正が必要な点
- **「綺麗に回すだけの人になっていくのかな」**: この気持ちはあったが、それだけではなく「いろいろかじっているけど、このまま進めて何者なんだ」という不安が大きかった
- **タッチポイントの順序**: YOUTRUST、note、Xの順序が違う

#### ✅ 正しい理解
- **時系列**: 
  1. 上司に相談（最初）
  2. 転職エージェントに登録（ルーダー）
  3. Xで転職情報を収集（転職の極意、人事系の人をフォロー）
  4. noteで未経験転職の経験談を見る
  5. Wantedlyで探す（以前から使っていた）
  6. スカウトメッセージをもらう（決定的な要因）
  7. 応募

- **タッチポイント**:
  - X（転職情報、人事系の人、転職の極意を発信している人）
  - note（未経験転職の経験談、人事・面接側の意見）
  - Wantedly（以前から使っていた）
  - スカウトメッセージ（決定的な要因）
  - リクナビ（登録していた）
  - Green（見ていたが、エージェント経由で使った）

### 3. Wantedlyに関する修正点

#### ✅ 重要な発見
- **スカウトメッセージが決定的**: 募集要項自体で刺さったというより、スカウトメッセージの文面が他のスカウト文と全然違う感じで面白いなと思った
- **最初は応募しなかった**: Wantedlyで探した時に見ていたが、「経験者じゃないとこれむずいだろうな」と思って応募まで進まなかった
- **「オープンポジション」が重要**: スカウトメッセージで「オープンポジションで未経験でもチャンスあるんだ」と知って応募

#### ✅ タイトルに必要な要素
- **「マーケティング」**: あった方がより目に留まる
- **「未経験」**: 重要なワード
- **「オペレーション」**: 入っていたらもっと目に留まる
- **「自分がやってきたことを活かしつつ、自分が興味があることも学びながらできるのか」**: これが魅力的

#### ❌ 刺さらない要素
- **「成長」「事業拡大」などのビッグワード**: マイベストと同じような感じだったので、そこに対しての物足りなさとかはなかった
- **「カオス」**: ネガティブに感じる可能性がある。「やるべきことがいっぱいある」の方がいい
- **「SN側」「SEO側」などの専門用語**: スッと飛ばしてしまう

### 4. 職種名に関する修正点

#### ✅ 推奨
- **「マーケティングオペレーション」**: やりたいこととできることを重ね合わせという感じで、結構いい

#### ❌ 避けるべき
- **「PdM」**: 絶対わからない
- **「コンサルタントアシスタント」**: 微妙

### 5. その他の重要なポイント

- **「自分が何者かが見つかる場所」**: 成長だったりとかに結びついているといい
- **「やるべきことがいっぱいある」**: 「カオス」より前向きな表現
- **「0→1を何かをみんなで作るのはすごく好きで楽しい」**: ただし「0→1を何かやりたいという欲望はそんなないかも」。チャンスがあったらもちろんやりたいという感じ

## 改修後のペルソナ・ジャーニー・Wantedly運用方針

### ペルソナ改修方針

1. **課題意識の修正**
   - 「成長機会と適切な報酬を両立できない」は維持
   - 「独立採算制度が決め手」→「面白い制度だと思ったが、それほど大きな決め手ではなかった」に修正
   - 「何者でもなかった」状態で、ビジネスパーソンとしてどこまでできるのか、何ができるのかわからないという漠然とした不安を追加

2. **動機の修正**
   - 「明確な目標像」→「現状への漠然とした不安」に修正
   - 「このままで私何が残るんだろう」という不安を追加

### ジャーニー改修方針

1. **段階1（日常・モヤモヤ期）の修正**
   - 「綺麗に回すだけの人になっていくのかな」→「いろいろかじっているけど、このまま進めて何者なんだ」という不安が大きかった
   - 「4年間勤めてある程度経験した・やりきった感があったところで、ここからどうしようと考えた時に、このままで私何が残るんだろうという不安を感じ始めた」

2. **タッチポイントの修正**
   - 時系列を正しく反映：上司に相談 → 転職エージェント → X → note → Wantedly → スカウトメッセージ
   - YOUTRUSTは削除（登録はしていたが、転職自体には使っていなかった）

3. **段階2（課題認識・探索期）の修正**
   - 「未経験でも、私の武器を活かしながら挑戦できる場所はないか？」を維持
   - 「マーケティングに興味があり、マーケティングを調べながら自分の活かせる場所を探していた」を追加

### Wantedly運用方針

1. **タイトル案の改修**
   - 「マーケティング」「未経験」「オペレーション」を含める
   - 「自分がやってきたことを活かしつつ、自分が興味があることも学びながらできるのか」というメッセージを含める
   - 「成長」「事業拡大」などのビッグワードは避ける
   - 「カオス」という言葉は避け、「やるべきことがいっぱいある」などの前向きな表現を使う

2. **スカウトメッセージの重要性**
   - スカウトメッセージの文面が決定的な要因だった
   - 「オープンポジションで未経験でもチャンスあるんだ」というメッセージを含める
   - 他のスカウト文と全然違う感じで面白いなと思える内容にする

3. **職種名**
   - 「マーケティングオペレーション」を推奨
   - 「PdM」は避ける

4. **募集要項の内容**
   - 「総合的にやっていて、成果を出すための取り組み」というニュアンス
   - 「一つの手段に取られるとかではなく、いろいろできるのか、いろんな形で学べるのかな」というメッセージ
   - 「自分がどうなれるかがわからない」状態なので、そこに対してアンサーがあるような内容
