# Tantan キーワード戦略サマリ（統合版）

## 1. 確実に押さえておくべき「コアキーワード」
3つのペルソナ（マーケター、事業会社リーダー、フリーランス）全員に共通して出現し、かつTantanの価値（シンプル、外部連携、脱・転記）に直結する**最重要キーワード群**です。これらはSEOの柱となり、LPのメイン訴求にも使うべきワードです。

### Tier 1：最優先攻略（ビッグワード・準ビッグワード）
> 検索ボリュームがあり、かつ全員が抱える課題のど真ん中。ここを取れれば認知が広がる。

- **タスク管理** / **タスク管理ツール**
- **進捗管理** / **進行管理**
- **ToDoリスト** / **TODOアプリ**
- **タスク漏れ** / **抜け漏れ** / **見落とし**
- **効率化** / **業務効率化**
- **依頼管理** / **依頼フォーム**

### Tier 2：Tantan独自の勝ち筋（独自の強み・解決策）
> 競合ツール（Asana/Trello等）に対する差別化要素であり、検索意図がTantanの機能と合致するもの。

- **URL共有** / **アカウント不要** / **登録不要**
- **クライアント共有** / **外部連携** / **社外 共有**
- **シンプル** / **簡単** / **直感的**
- **転記不要** / **一元管理** / **一元化**
- **言った言わない** / **エビデンス**
- **Slack連携** / **チャット連携**
- **スマホ対応** / **マルチデバイス**

## 4. 分析からの示唆・アクションプラン

### 「外部連携の容易さ」が最大の共通ニーズ
3者とも**「自分以外の人（クライアント、業務委託、部下）」**とのタスクのやり取りに課題を感じています。「アカウント登録不要」「URL共有だけ」というキーワードは、全ペルソナに刺さる**最強のキラーワード**です。

### 「Slack/Chatwork」は敵ではなく入り口
「Slack 通知 多すぎ」「Chatwork タスク 使いにくい」といった、既存コミュニケーションツールの**不満系キーワード**は大きなチャンスです。「Slackと連携して通知を整理する」「ChatworkのタスクをURL化する」といった訴求が有効です。

### 「転記」という行為自体を敵にする
「メールから転記」「チャットからコピペ」という作業への不満は共通しています。**「転記不要」「自動化」「依頼フォーム」**というキーワードで、このペインを解決できることを強く打ち出すべきです。
