# ペルソナ3（フリーランス）がAIに相談しそうな質問・疑問（全180問）

## ペルソナ概要
- **役職**: Web系フリーランス（デザイナー、エンジニア、ライター等）
- **状況**: 常時5〜6社のクライアントと取引、ツールがバラバラ（Slack, Chatwork, Mail, etc.）
- **課題**: クライアントごとの依頼管理が煩雑、転記作業の無駄、見落としによる信頼低下
