### 5. 一般KWでCVが獲得できない要因の整理

**検証した仮説と結果**

```
仮説1：予算が足りない
  → 対応：5倍に増額（285万円）
  → 結果：効果なし

仮説2：インプレッションシェアが低い
  → 対応：20%まで引き上げ
  → 結果：効果なし

仮説3：キーワード選定が悪い
  → 対応：CV実績のあるKWに絞り込み
  → 結果：効果なし

仮説4：LP（CVR）が悪い
  → 対応：フォーム改善を実施
  → 結果：効果なし

仮説5：品質スコアが低い
  → 状況：1〜3と判明
  → 対応：★未実施★
```

**現時点での見解**

**問題1：CPCの高騰（品質スコアの問題）**
品質スコアが1〜3と低いため：
- CPCが高騰している（実際に2,381円まで上昇）
- 同じ予算でも獲得できるクリック数が減る
- 広告の表示順位も不利になる

**問題2：CVR 0%（LPと検索意図のミスマッチ）**
479クリックあってCV 0件という事実は、品質スコアだけでは説明できない。
- 「脆弱性診断」「セキュリティ診断」で検索するユーザーの検索意図
- 現在のLP（AeyeScanの訴求）
- この2つがマッチしていない可能性が高い

→ **品質スコア改善とLP改善は両輪で必要**

**競合環境について**
品質スコアを改善しても、競合との予算差（3〜10倍）は埋まらない。
ただし、品質スコアが改善すれば同じ予算でより効率的に戦える。
問題は「効率を上げても勝てるのか」という点。

### 6. 判断の分岐点

**選択肢A：一般KWの改善を継続**

```
やること：
- 品質スコアの改善
  - 広告文の見直し（KWとの関連性を高める）
  - LPの改善（検索意図に合った内容に）
  - アカウント構造の見直し

必要な期間：
- 品質スコアの改善には2〜3ヶ月の検証が必要

期待できる効果：
- 品質スコアが上がればCPCが下がる
- CPCが下がれば同じ予算でより多くのクリックが獲得できる
- CVRも改善する可能性がある

リスク：
- 改善しても競合との予算差（3〜10倍）は埋まらない
- 2〜3ヶ月かけても成果が出ない可能性がある
- その間も一定の広告費がかかる
```

```
やること：
- 一般KWへの投資を停止または最小限に

- CPAを安定させられる

デメリット：
- 一般KWでの新規獲得は見送ることになる
- リード目標58件の達成は別の方法で補う必要がある
```

### 7. 撤退ラインの提案

**継続判断の基準（段階的に評価）**

| 期限 | 確認項目 | 達成基準 | 未達の場合 |
|------|----------|----------|------------|
| 2月末 | 品質スコア | 5以上に改善 | 継続困難と判断 |
| 3月末 | CVR | 0.3%以上を達成 | 一般KWは縮小 |

**なぜ2段階にするか**
- 品質スコアの改善には通常2〜3ヶ月かかる
- まず2月末で「改善の兆候があるか」を確認
- 兆候があれば3月末まで継続し、CVRで最終判断

**補足**
- 品質スコア5以上：CPCを適正水準（1,000円以下目安）に下げられる
- CVR 0.3%：最低限このレベルがないと広告として成立しない
- 両方を満たさない場合、一般KWでの獲得は現実的ではないと判断し、
  浮いた予算を別チャネル（Part 2で検討）に振り向ける

### 2. 即商談につながるチャネルの特定

BigQueryと商談一覧を分析した結果、「即商談」につながるチャネルが明確になった。

**チャネル別の特性**
