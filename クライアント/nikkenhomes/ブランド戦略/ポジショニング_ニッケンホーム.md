# ポジショニング分析レポート：ニッケンホーム

## エグゼクティブサマリー

### 現在の市場における各社の位置づけ
愛知・岐阜エリアの注文住宅市場において、ニッケンホームは「高性能×コストパフォーマンス」を軸に、数値で証明できる品質を訴求。一方、競合neieは「建築士との対話×デザイン性」を軸に、情緒的価値とプロセスの丁寧さを訴求している。

### ニッケンホームの相対的なポジション
- **強み**：性能数値の明確な開示、豊富な実績（3,800棟以上）、ZEH普及率90%、コストパフォーマンス訴求
- **独自性**：福祉施設ノウハウを活かしたバリアフリー設計

### 顧客から見た差別化ポイント
- 「性能重視×コスト意識の高い顧客」にはニッケンホームが選ばれやすい
- 「デザイン重視×対話プロセス重視の顧客」にはneieが選ばれやすい

※ウェブリサーチによる事実に基づく分析

## ポジショニング分析

### Map 1: 性能訴求 × 価格訴求

```
        性能重視（数値明示）
              ↑
              |
    [ニッケンホーム]
              |
              |
低価格訴求 ←----+----→ 高価格/価値訴求
              |
              |
        [neie]
              |
              ↓
        デザイン/情緒重視
```

**分析**：
- **ニッケンホーム**：HEAT20基準（G1〜G3）、UA値、C値、耐震等級3などの性能数値を明確に開示。「コストパフォーマンス」を前面に訴求し、グレード別の選択肢を提供。
- **neie**：性能数値の開示は少なく、「ていねい」「対話」「建築士」などの情緒的価値を訴求。価格情報は非公開だが、建築士直接対応の体制から中〜高価格帯と推定。

※参照ページ：https://www.nikkenhomes.co.jp/grades/, https://neie.jp/

### Map 2: 対象顧客層 × サポート体制

```
        ファミリー・子育て層
              ↑
              |
    [ニッケンホーム]
              |
              |
効率重視 ←----+----→ 対話重視（時間をかける）
              |
              |
           [neie]
              |
              ↓
        こだわり層・デザイン志向
```

**分析**：
- **ニッケンホーム**：子育て世代向けの施工事例が多数。「納得するまで打ち合わせ」を訴求しつつ、グレード選択による効率的な進め方も可能。
- **neie**：「一家族ひと家族に暮らしがある」という個別対応、建築士が初回から対応、8つの設計プロセス要素など、時間をかけた対話を重視。

※参照ページ：https://www.nikkenhomes.co.jp/works/, https://www.nikkenhomes.co.jp/voice/, https://neie.jp/about/, https://neie.jp/renovation/

- **neie**：具体的な実績数は非公開だが、創業101年の歴史、設計プロセスの8つの要素、建築士直接対応体制など、プロセスの透明性で信頼性を証明。

※参照ページ：https://www.nikkenhomes.co.jp/, https://www.nikkenhomes.co.jp/concept/, https://neie.jp/about/, https://neie.jp/renovation/

## 顧客の意思決定における各社の位置づけ

### 選ばれる理由の分析

- **性能重視の顧客** → ニッケンホーム
  - 理由：UA値、C値、耐震等級などの具体的数値が明示されている
  - ZEH普及率90%という客観的実績がある
※参照ページ：https://www.nikkenhomes.co.jp/grades/

- **コストパフォーマンス重視の顧客** → ニッケンホーム
  - 理由：「コストパフォーマンスが良かった」という顧客の声
  - グレード選択による予算に応じた対応が可能
※参照ページ：https://www.nikkenhomes.co.jp/voice/, https://www.nikkenhomes.co.jp/grades/

- **デザイン・対話重視の顧客** → neie
  - 理由：建築士が初回から対応する体制
  - 「ていねい」を前面に出した情緒的訴求
※参照ページ：https://neie.jp/, https://neie.jp/about/

- **リノベーション検討顧客** → neie
  - 理由：空間全体のリノベーションサービスを展開
  - ニッケンホームはリノベーション非対応
※参照ページ：https://neie.jp/renovation/

## カスタマージャーニー作成のためのポジショニング情報

### ニッケンホームが選ばれるシナリオ

- **顧客タイプ**：性能とコストのバランスを重視する子育て世代
※参照ページ：https://www.nikkenhomes.co.jp/voice/

- **重視する価値**：
  - 断熱・気密性能による快適性と光熱費削減
  - 耐震性能による家族の安全
  - 長期的なコストパフォーマンス
※参照ページ：https://www.nikkenhomes.co.jp/grades/, https://www.nikkenhomes.co.jp/concept/

- **ニッケンホームの訴求点**：
  - HEAT20基準準拠の具体的な性能数値
  - 3,800棟以上の施工実績
  - ZEH普及率90%という客観的データ
  - 「誠実」「納得するまで」の対応姿勢
※参照ページ：https://www.nikkenhomes.co.jp/grades/, https://www.nikkenhomes.co.jp/voice/

### 各段階での差別化ポイント

- **認知段階**：「高性能住宅」「ZEH」「コストパフォーマンス」で検索時に優位
※参照ページ：https://www.nikkenhomes.co.jp/

- **検討段階**：グレード別スペック比較表で他社との性能比較が容易
※参照ページ：https://www.nikkenhomes.co.jp/grades/

- **決定段階**：「全てにおいてコストパフォーマンスが良かった」という顧客の声が後押し
※参照ページ：https://www.nikkenhomes.co.jp/voice/

### 競合との相対的ポジション

- **ニッケンホームの強み領域**：
  - 性能数値の明確な開示
  - 豊富な施工実績と受賞歴
  - ZEH対応力
  - コストパフォーマンス訴求

- **neieの強み領域**：
  - 建築士直接対応体制
  - デザイン性・情緒的価値
  - リノベーション対応
  - 設計プロセスの透明性

- **空白市場**：
  - 「高性能×デザイン性×対話プロセス」の全てを高いレベルで実現する領域
  - 建築士の顔が見える高性能住宅

- **強化すべき領域**：
  - 設計プロセスの可視化
  - 建築士・設計士の顔が見える訴求
  - デザイン性のアピール

### コミュニケーションの方向性

- **メインメッセージ**：「50年先も、性能も、コストも、安心」
※参照ページ：https://www.nikkenhomes.co.jp/

- **サブメッセージ**：
  - 「HEAT20基準で証明する、本当の快適」
  - 「3,800棟の実績が裏付ける、確かな品質」
  - 「誠実に、納得するまで」
※参照ページ：https://www.nikkenhomes.co.jp/grades/, https://www.nikkenhomes.co.jp/voice/

- **避けるべき訴求**：
  - デザイン性のみの訴求（neieが強い領域）
  - プロセスの情緒的価値のみの訴求

- **強化すべき訴求**：
  - 数値で証明できる性能
  - 実績に基づく信頼性
  - 長期的なコストメリット
  - 福祉施設ノウハウを活かしたユニバーサルデザイン
