# 寺倉ナレッジ

## 概要
寺倉大史（somehiko@moltsinc.co.jp）の会議記録からナレッジを蓄積するシステム

## 目的
- 寺倉さんのAIハック・経営判断をリアルタイムでキャッチアップ
- 会議で出た知見を検索可能な形で保存
- テーマ別にナレッジを統合

## フォルダ構成
```
寺倉ナレッジ/
├── CLAUDE.md（このファイル）
├── 会議一覧/        # 会議ごとのサマリー
│   └── YYYY-MM-DD_会議名.md
└── サマリー/        # テーマ別の統合ナレッジ
    ├── AI活用.md
    ├── SEO_LLMO.md
    ├── 採用.md
    └── プロダクト開発.md
```

## 使い方

### 会議を取得・保存
「寺倉MTG取得」→ Firefliesから直近の会議を取得して保存

### テーマ別にまとめる
「寺倉ナレッジまとめて」→ 会議一覧からテーマ別にサマリー作成

### 特定テーマを検索
「寺倉さんLLMOについて何か言ってた？」→ 関連会議を検索して回答
