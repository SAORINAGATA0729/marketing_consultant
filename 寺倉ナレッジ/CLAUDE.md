# 寺倉ナレッジ

## 概要
寺倉大史（somehiko@moltsinc.co.jp）の会議記録からナレッジを蓄積するシステム

## 目的
- 寺倉さんのAIハック・経営判断をリアルタイムでキャッチアップ
- 会議で出た知見を検索可能な形で保存
- テーマ別にナレッジを統合

## フォルダ構成
```
寺倉ナレッジ/
├── CLAUDE.md（このファイル）
├── 会議一覧/        # 会議ごとのサマリー（50件）
│   └── YYYY-MM-DD_会議名.md
└── サマリー/        # テーマ別の統合ナレッジ
    ├── AI活用.md
    ├── SEO_LLMO.md
    ├── 採用.md
    └── プロダクト開発.md
```

## 使い方

### 会議を取得・保存
「寺倉MTG取得」または `/terakura-sync` → Firefliesから直近の会議を取得して保存

### テーマ別にまとめる
「寺倉ナレッジまとめて」→ 会議一覧からテーマ別にサマリー作成

### 特定テーマを検索
「寺倉さんLLMOについて何か言ってた？」→ 関連会議を検索して回答

## 寺倉さんの最近のキーワード

### AI・開発
- **クロードコード一本化**: 2月末までに全ツールをクロードコードに統一
- **並列実装**: 全社員がAIで並列開発できる体制（3-4月目標）
- **スキル図**: 業務をスキル化してツール化
- **AIコーディング支援**: 全機能の初期実装をAIに任せる

### マーケティング・SEO
- **LLMO**: AIに選ばれるコンテンツ戦略
- **FAQ作成**: 1問1ページ、300文字の端的な回答
- **ドメイン統合**: SEO評価の分散リスク回避

### 組織・カルチャー
- **サーキット化**: プロジェクト設計の転換
- **生産性5倍→組織で1倍問題**: 個人のAI活用は進むが組織全体は遅れる
- **伝説を作る意識**: ミッション達成への主体性
